

### Modelagem de Dados para Banco de Dados.
 
  
##*Aula 1:* _Introdução à Modelagem de dados - Parte 1_


Modelagem: 

	Representação
	Modelo 
	Referência, que irã servir de base para outros cenários.


Definição de Modelagem: 

Possui foco na descrição e relacionamento dos e
lementos que compõem a representação do contexto(mini-mundo).


Processos da Modelagem: 

	Mini-mundo -> Delimita o contexto dos dados
	Alto nível -> Requisitos para criação do modelo
	Esquema    -> Definindo estrutura relacional
	SGBD       -> Implementando - Criando o DB

E o Esquema? 

	Modelos de alto nível: 

	- Entidade-Relacionamento
	- UML(Unfied Modeling Language), muito usada, acabando entrando em desuso, 
	por causa da abordagem mais voltada ao efeito cascata, as Metodolias Ágeis
	assumiram o lugar da UML, sendo mais utilizada.
